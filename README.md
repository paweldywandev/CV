# Paweł Dywan - CV

This repository contains my professional CV in both English and Polish versions, built using [RenderCV](https://rendercv.com/).

## Files

- `Pawel_Dywan_CV.yaml` - English version of the CV
- `Pawel_Dywan_CV_PL.yaml` - Polish version of the CV (Polska wersja CV)

## Generating PDF

To generate PDF files from the YAML sources, you need to have RenderCV installed:

```bash
pip install rendercv
```

Then run:

```bash
# Generate English version
rendercv render Pawel_Dywan_CV.yaml

# Generate Polish version
rendercv render Pawel_Dywan_CV_PL.yaml
```

The generated files will be placed in the `rendercv_output/` directory.

## About RenderCV

RenderCV is a LaTeX-based CV/resume framework that allows you to define your CV in a YAML file and automatically generates a professionally formatted PDF.

For more information, visit the [RenderCV documentation](https://docs.rendercv.com).
